Directory Structure & Contents

The USB20HR IP core has following directory structure.

 

documents

Contains Technical Documents such as user guide, tutorial and API guide for the USB20HR IP Core. You can refer all the documents by following the link Documentation.  

hardware

Contains HAL driver, encrypted USB20HR IP Core design files, demonstrations files using the core, ready to use reference designs for the core and precompiled library for simulation

 

software

Contains SLSUSB driver files for windows and linux with application examples.

embedded

Contains source files for applications with Nios II.

readme.htm

This file.

 

 

 

Note:

The folder name is <path>/usb20hr_<lic_type>.  Here, “path” is the IP Core default installation path.  Default path is C:/sls.  The "lic_type" is variable.  “lic_type” is IP Core license type – ocp_eval or - oc_full. Example: usb20hr_ocp_eval means “lic_type” is evaluation.
