How to Get the License

You can have two types of licence. Full and Eval (OpenCore Plus)

 

For the full version, System Level Solutions provides a license file which allows the user to integrate the USB20HR IP through the SOPC builder. In this license file the programming file generation is supported and .sof & .pof files are generated.  This license is valid up to 1 year.

 

For eval version, System Level Solutions provides a license through email. To get the eval license follow a steps as shown in readme file given inside license folder.  In this license file the time limited programming file generation is supported.

 

System Level Solutions is responsible for the behavior of the USB20HR Core and can also support implementation issues of the core in the Altera environment!
