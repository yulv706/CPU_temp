hardware

The hardware folder contains following:

 

component
Contains HAL driver as well as encrypted USB20HR IP Core design files

HAL

Contains Driver for USB20HR IP Core

inc

Contains Header file to declare driver API

src
Contains Driver Object File

hdl

Contains encrypted IP Core files

inc
Contains header file to declare USB20HR register interface


demo
Contains demonstrations files designed using USB20HR IP core. To run demonstrations, refer the readme files located inside the respective folder.

ulpi

Contains demonstrations using ulpi interface

massstorage_ulpi_2C35c

Mass storage demonstration for Nios II Development Kit, Cyclone II (2C35Revc) using ULPI Interface

portinterface_ulpi_2c35c

Port Interface demonstration for Nios II Development Kit, Cyclone II (2C35Revc) using ULPI Interface

streaming_ulpi_2c35c

 

bulk_in

Streaming (bulk_in) demonstration for Nios II Development Kit, Cyclone II (2C35Revc)

bulk_out

Streaming (bulk_out) demonstration for Nios II Development Kit, Cyclone II (2C35Revc)
