software

The software folder contains following:

 

driver
Contains SLSUSB driver for linux as well as windows

linux

SLSUSB synchronous driver is designed for the Linux kernel version 2.6, upto 2.6.18.

win2k_xp_x32

SLSUSB Driver files, i.e. System files (*.sys) & INF file.


example

linux

The SLSUSBClient application gives user an idea about how to perform read/write operation on SLSUSB
                     supported devices on Linux machine

windows

Contains demonstrations using utmi interface

rwusb_c_sharp

Contains c# code for making a console base USB application

rwusb_vc_pp

Contains vc++6.0 code for making a console base USB application


library

c_sharp_dotnet

Contains *.dll for C# programming

vc_pp

Contains .h header file,.lib(library file) and .dll for VC++ programming


utilities

enumdataeditor

This application is used to modify enumeration data for USB20HR device IP core.This application
                                modifies the content of HEX file used during Enumeration of the device.

portinterface

Test to show USB to & from Xfer on any other peripheral in the design.

Readme.txt

Shows steps how to use PortInterface.exe

usbview

USBView application for  windows 2000/XP to show USB  detection & check for driver load.
