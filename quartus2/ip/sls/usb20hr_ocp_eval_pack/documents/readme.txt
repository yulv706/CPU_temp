As you have downloaded USB20HR evaluation package you have to use
evaluation license for compilation in Quartus II software.

To get evaluation license log on to www.slscorp.com and go to  licensing section and mention 
the NIC number of the PC for which you want to have the license.


Please follow the following steps to get the NIC number:

    - Open up Command Prompt.
    - Type "ipconfig/all".
    - The Physical Address will give you 12 digit no, which 
      is nothing but your NIC number.
