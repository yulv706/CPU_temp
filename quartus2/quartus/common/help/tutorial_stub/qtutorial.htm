Install the Quartus II Interactive Tutorial

 

The Quartus II Interactive Tutorial is not installed on this computer. You can choose whether to install the tutorial at the time of Quartus II installation, or you can subsequently run a Custom Installation and turn on the Tutorial option to install the tutorial separately.

The modules of the interactive tutorial teach you how to use the basic features of the Quartus II design software, including design entry, compilation, timing analysis, simulation, and programming. This tutorial includes audio and Flash animation components, and is best experienced with a sound card and speakers and at least 1024x768 display resolution.

Once you start the tutorial, you can jump immediately to any tutorial module by clicking the Contents button. Once you start a tutorial module, you can click the ShowMe, GuideMe, or TestMe buttons at any time to jump directly to that tutorial mode that best suits your learning style.

To use the tutorial, rerun the Quartus II installation program and select the Tutorial component in the Custom Installation. For more information about the Quartus II software, refer to Quartus II Online Help.
